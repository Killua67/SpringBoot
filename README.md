# SpringBoot
自己学习SpringBoot时的demo

**包含内容**
1. web框架(用于web开发)
2. Redis(缓存)
3. thymeleaf(前端模板)
4. spring data jpa (hibernate)
5. RabbitMQ(消息队列)
6. Email(邮件服务)
7. shiro(登录认证和权限管理)

代码中有很多注释，仅供学习使用
